#### 排序

1. 插入法

```
int[] a = {1, 3, 5, 4, 62, 467, 12, 0};
for (int i = 1; i < a.length; i++) {
    int tis = a[i];
    int index =i;
    for (int j = i; j > 0 && tis< a[j - 1]; j--) {
        //当前数等于前面一个数，前面的数往后移动了
        a[j] = a[j-1];
        index =j-1;
    }
    a[index] = tis;
}
System.out.println(Arrays.toString(a));
```
