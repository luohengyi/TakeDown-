## linux

####  系统命令	

##### 包管理类

1. 安装命令 yum install
2. 下载文件 wget +地址   (yum install wget安装wget命令)
3. 软件卸载 yum remove xxx
4. 软件模糊搜索：yum serach xxx

##### 进程类

1. 查看进程：ps -ef|grep +程序名称   
2. 查看所有进程：ps -ef (ps -ef | grep nginx  查看nginx的进程
3. 杀死进程
   1. pkill nginx 杀死nginx进程
   2. kill 6000 杀死进程号

##### 文件类

1. 新建文件： touch +文件名
2. 查看文件：cat +文件名
   1.  cat -b 展示行号
   2. tac 逆序展示
   3. more 分页查看,空格键翻页
3. 查看文件列表：ls （ls -al查看文件的详细信息）
